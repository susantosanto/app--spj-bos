Import Data BKU

Pilih metode import untuk memulai ekstraksi transaksi otomatis ke dalam sistem SPJ Flow

Google Drive

Import langsung dari file Excel atau Google Sheets di Drive Anda

Rekomendasi

File Lokal

Upload file Excel dari komputer atau perangkat penyimpanan lokal

Persyaratan File BKU

Format File

Excel (.xlsx) atau Google Sheets dengan kolom: Tanggal, No Bukti, Uraian, Pengeluaran

Keamanan Data

Data diproses secara lokal, tidak ada data yang dikirim ke server eksternal

Proses Otomatis

Ekstraksi otomatis dalam hitungan detik, siap untuk dicetak menjadi dokumen SPJ
