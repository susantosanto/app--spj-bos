Panduan Format Excel BKU

Agar file BKU dapat dibaca oleh aplikasi, ikuti format berikut:

Format File yang Didukung

Google Sheets

Format terbaik! File tersimpan otomatis di Google Drive dan mudah dibaca.

Excel (.xlsx)

File Excel akan dikonversi otomatis ke Google Sheets saat diunggah.

Struktur Kolom Excel

Kolom Nama Kolom Contoh Isi Wajib Diisi?
A Tanggal 01/01/2025 ✓ Ya
B Kode Rekening 5.2.1.02.01 Opsional
C Uraian Belanja Makan & Snack ✓ Ya
D No. Bukti 001/SD-BOS/2025 ✓ Ya
E Penerima RM. Family Opsional
F Pemasukan - Opsional
G Pengeluaran 1.500.000 ✓ Ya
Catatan Penting:
  • Kolom A (Tanggal), C (Uraian), D (No. Bukti), dan G (Pengeluaran) WAJIB diisi!
  • Aplikasi hanya akan membaca transaksi dengan Pengeluaran > 0 dan No. Bukti tidak kosong.
  • Transaksi dengan No. Bukti mengandung kata "PPH", "PPN", atau "PAJAK" akan diabaikan (karena itu pajak, bukan pengeluaran).
  • Format tanggal: DD/MM/YYYY (contoh: 01/01/2025)
  • Format angka: gunakan titik sebagai pemisah ribuan (contoh: 1.500.000)

Tips Tambahan

1
Gunakan Google Sheets

Google Sheets lebih mudah dibaca aplikasi dan tersimpan otomatis di Drive. Cukup buat spreadsheet baru dengan nama "BKU 2025".

2
Baris Pertama adalah Header

Pastikan baris pertama berisi nama kolom (Tanggal, Kode Rekening, Uraian, dll). Aplikasi akan mulai membaca dari baris kedua.

3
Nama File Mengandung "BKU" atau "ARKAS"

Aplikasi mencari file dengan kata kunci "BKU" atau "ARKAS" di Google Drive. Contoh: "BKU JANUARI 2025", "ARKAS 2025".

4
Kosongkan Kolom Pemasukan

Aplikasi ini fokus pada pengeluaran. Kolom pemasukan (F) bisa dikosongkan atau diisi dengan "-".
